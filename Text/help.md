*Сообщения могут отправлять только зарегистрированные пользователи*
```
* Для проверки пинга введи:
    /ping 0.0.0.0

* Для проверки запущенного драйвера набрать:
    /ps 0.0.0.0

* Для перезагрузки КШ(М):
    /reboot 0.0.0.0

* Для чтения лога (аналог команде tail -f <***.log>):
    /log 0.0.0.0

* Для поиска текста в названии объекта:
    /find текст запроса
    (пример: /find база)

* Для объекта по номеру ID:
    /id номер id
    (пример: /id 70)

* Для объекта по номеру КШ(М):
    /kshm номер КШ(М)
    (пример: /kshm 2888)
```
_Вместо 0.0.0.0 необходимо ввести проверяемый IP_
